## Vanilla JS To Do Kanban Board Project ##

### Summary ###
Vanilla JS로 To Do List Kanban Board 구현하기

### Tech Skill ###
- Vanilla Javascript
- Webpack, Babel
- Sass
- localStorage

### Funtionality ###
- 카드 추가
  - 입력폼 validation check
- 카드 수정
- 카드 삭제
- 카드 리스트 우선순위별 정렬
- 카드 drag & drop

### Run Local Server ###
```text
    npm run dev-server
```
